// Instructions to disable MySQL Default Stopwords //
// To be run using root access //

-> In the mysql config file, under the [mysqld] heading add the following lines

innodb_ft_enable_stopword = cmfive/my_stopwords
innodb_ft_server_stopword_table = 'cmfive/my_stopwords'

-> Save the file
-> Restart MySQL server

-> Then run the following Quereys
ALTER TABLE object_index DROP INDEX content; CREATE FULLTEXT INDEX content ON object_index(content);

-> The default stopword table has now been overidden!
