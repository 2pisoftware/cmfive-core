// Instructions to disable MySQL Default Stopwords //
// To be run using root access //

-> In the mysql config file, under the [mysqld] heading add the following lines

ft_min_word_len=1
ft_stopword_file = ""

// this will disable all mysql default stopwords. To re-enable the default stopwords comment the line out by adding a hash as seen below. 

# ft_stopword_file = ""


-> Restart MySQL server
-> Repair Search Table by querying the following 

REPAIR TABLE object_index QUICK;
